# Pattern Generator and Number Analyzer

## Description

This is a simple Python program that:

* Generates a star (*) pattern
* Analyzes numbers in a given range

---

## Features

* Generate star patterns
* Find even and odd numbers
* Count total numbers
* Calculate sum

---

## How to Run

1. Install Python
2. Save the file as `program.py`
3. Run the program:

   ```
   python program.py
   ```

---

## Menu

```
1. Generate a Pattern
2. Analyze a Range of Numbers
3. Exit
```

---

## Example Output

Pattern:

```
*
**
***
```

Number Analysis:

```
Even numbers: [2, 4]
Odd numbers: [1, 3]
Total numbers: 4
Sum: 10
```
